Detailed Description

Platform dependent system implementation.

To watch all tcat::pal notifications handlers you may run the dice cmd line tool (you can run it in parallel to your application - to debug notification handling) with the tracing (+t) and/or logging (+l) enabled: 

dice +t +l sg
Exceptions:
Besides internal exceptions a dice::xptn_driver_version will be thrown at construction time if the installed driver version is not compatible.

The application wide singleton tcat::dice::system instanciates tcat::dice::busses as reactions to OS device arrival/removal notifications. As far as a driver is installed the tcat::dice::system will create.
